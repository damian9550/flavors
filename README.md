flavors
=======

Flavors is library of GPU Trie structures for fast parallel lookup. It provides facilities to build Tries with changing bit stride and search them on the GPU.



## Building
```sh
#To build library
make

#To build samples
make sample
```

## Running samples
```sh
./bin/basicKeysSample

./bin/basicMasksSample
```
